# Portfolio — Hoto Tajima

Personal portfolio site built with React + Vite, featuring a liquid-glass design system, Framer Motion animations, and a custom canvas-based orbit/space background.

## Stack

- React 18 + Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons
- Custom canvas animation (orbit/space background)
- Google Fonts: Instrument Serif, Barlow

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173/ in your browser.

## Build

```bash
npm run build
npm run preview
```

## Sections

- **Home** — Hero with orbit background and stat cards
- **About** — Profile and highlights (JOP No.1, NCAA Div.2, Temple Univ., LLC, CEO)
- **Experience** — Timeline of roles
- **Projects** — Cards for each product (Orbito-ai & hikari-houto)
- **Skills** — Technical and leadership skills
- **Contact** — Email & GitHub links

## Deployment

Designed for Vercel — push to GitHub, import the repo on Vercel, and deploy.
